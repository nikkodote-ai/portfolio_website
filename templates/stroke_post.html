Stroke Prediction



{{wtf.quick_form(form)}}

This is not intended to be used for real-world medical diagnosis.

Using Decision Tree Classifier by Sci-Kit Learn and training the data with Kaggle datasets, the probability of having stroke is predicted according to the user input. Although the classifier has high accuracy (>90%), it has a low f1 score. This imbalance means that there is a high accuracy for predicting negative results but very low accuracy for predicting a correct positive result.
